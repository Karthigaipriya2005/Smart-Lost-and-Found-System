Smart-Lost-and-Found-System

📌 Overview
The Smart Lost and Found Item Recovery System is a web-based platform designed to efficiently manage the process of reporting, tracking, and recovering lost or found items within an institution or community.

This system replaces traditional manual tracking methods with a centralized, secure, and scalable digital solution, enabling seamless communication between users and administrators.

🚀 Features
🔐 User Registration & Login with Authentication
📧 Email Verification & Confirmation System
🔍 Report Lost Items with details and images
📦 Report Found Items
🆔 Unique ID generation for each item report
📊 Track status of submitted items
🛠 Admin Panel for verification and management
🔄 Password Recovery System
🖼 Optional Image Upload for items

📊 System Architecture
The system is structured using:
🔹 Key Entities:
USER
LOGIN
ITEM
LOST_ITEM
FOUND_ITEM
ADMIN

Improves efficiency in item tracking
Ensures transparency in reporting
